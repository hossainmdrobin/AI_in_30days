# 🚀 30 Days of AI Projects

This repository documents my **30-day journey of learning and building AI applications**.  
During this challenge, I explored how to use **Large Language Models (LLMs)**, automation, and AI APIs to create practical tools.

The goal of this journey was to:

- Understand how modern AI tools work
- Learn how to integrate AI APIs into applications
- Build small but practical AI-powered tools
- Prepare to integrate AI features into real products

Some projects were **skipped intentionally** because they were very similar in structure or functionality to previous ones. Instead of repeating nearly identical implementations, I focused on learning the core concepts and moving forward to new ideas.

# 🧠 Technologies Used

- Python
- LLM APIs
- CLI-based applications
- Prompt Engineering
- Automation

Primary AI services explored:

- Groq API
- LLM models such as Llama
